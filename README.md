# Arduitouch MQTT UI

ArduiTouch MQTT UI is a MQTT Client und UI for the ArduiTouch with an ESP32.

## Installation

U need the following additional Librarys:

Adafruit_GFX

Adafruit_ILI9341

XPT2046_Touchscreen

DHTesp

Ticker  please install from https://github.com/bertmelis/Ticker-esp32
